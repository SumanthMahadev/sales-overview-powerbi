# Sales Overview Dashboard — Power BI

## Project Overview
An interactive single-page sales dashboard built in Power BI 
using Microsoft Financial Sample data covering 2013-2014.

## Skills Demonstrated
- Power Query — data cleaning and transformation
- DAX — 4 measures including SUM, DIVIDE, Profit Margin %
- Data visualisation — 9 visuals on one page
- Dashboard design — professional layout and formatting

## Visuals Included
- 4 KPI Cards — Total Sales, Total Profit, Profit Margin %, Units Sold
- Line Chart — Sales trend by Month
- Bar Chart — Sales by Country
- Donut Chart — Sales by Segment
- Gauge Chart — Profit Margin vs 20% target
- Product Performance Table with data bars
- Treemap — Sales by Country
- 3 Dynamic Slicers — Year, Segment, Country

## Tools Used
- Power BI Desktop
- Power Query (M language)
- DAX
- Microsoft Excel

## Dataset
Microsoft Financial Sample — freely available from Microsoft Learn
